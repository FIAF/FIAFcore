# FIAFcore

The FIAFcore ontology is primarily based on the FIAF [Moving Image Cataloguing Manual](https://www.fiafnet.org/pages/E-Resources/Cataloguing-Manual.html).   
Note that there are currently no class vocabularies, due to the belief that these should be defined in response to those present in the community, and not devised in isolation.
